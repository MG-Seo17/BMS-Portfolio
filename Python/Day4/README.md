1. 주석 : '# ~~' 을 주석으로 사용한다... 여러 행 걸친 주석은 없다. 그러나 ' """ ~ """ ' 을 본래의 사용법과 조금 다르게 여러 행 주석으로 사용하기도 한다.
2. 줄바꿈 : 행이 길어질 경우 줄바꿈을 이용하여 코드를 작성한다. WHY? 가독성을 위함이다. 
            백슬래쉬 (\를 이용한다.), 리스트나 사전의 요소를 줄바꿈을 하는 경우는 그냥 ENTER 키를 이용한다.
3. 인덴트 : 문장 앞 들여쓰기 하는 방식... C,C++처럼 {}를 통해 그룹화를 하지 않는다. 따라서 TAB이나 ENTER키를 이용하여 들여쓰기를 해준다.
            일반적으로는 ENTER 4번을 해준다.
            if 조건식 :
                명령1
                명령2
            명령 3
            # 조건 만족시 명령 1과 2가 실행되고 그렇지 않다면 명령 3이 실행이 된다
4. 조건식 평가 : TRUE or 0인 경우 조건 성립으로 간주한다.
            (1) 비교연산자 : 2개 값을 비교한 후 부울값을 반환, 튜플이나 리스트 비교시 0번쨰 값만 비교
                -->  a == b    #2개 값 동일시 TRUE
                     a != b    #2개 값 다르면 TRUE
                     a > b     #a가 크다면 TRUE
                     a >= b    #a가 b이상이면 TRUE
                     a < b     #a가 작다면 TRUE
                     a <= b    #a가 b 이하면 TRUE
                     a in b    #a가 b에 포함이면 TRUE
5. 부울 연산자 : 동시에 여러 값들을 비교하는 연산자이다. 
                 --> 조건식1 and 조건식2 : 2개다 참인 경우 TRUE
                     조건식1 or 조건식2 : 1개라도 TURE이면 TRUE
                     not 조건식1 : 조건식1과 반대의 부울값 반환
                     (BUT 리스트 대입 시 and의 경우는 리스트 2가 or는 리스트1이 not은 공백이면 TRUE 아니라면 FALSE이다.)
6.if 문 : 조건 충족시 명령을 실행
          --> if - else문, if문, if - else if(elif) 3개가 존재.
