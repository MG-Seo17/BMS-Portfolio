                 -->  a = [1, 2, 3]
                      b = a
                      a[2] = 9   # 이렇게 되면 a와 b 둘다 [1, 2, 9]를 가진다. Why? 같은 리스트를 참조하고 있기 때문이다.

                      따라서 b = a.copy() 를 b =a 대신 적어 리스트 자체를 복제하게 되면 b는 [1, 2, 3] a는 [1, 2, 9]가 된다.
