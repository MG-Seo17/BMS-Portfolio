## 공부한 내용

1. 리스트나 튜플을 다루는데 편리한 함수
      (1) len : 요소수를 반환. 즉, 함수에 요소가 몇 개인가를 판단
      (2) copy : 리스트를 복제하고 싶을 때 사용.(튜플에는 사용을 하지 않는다.)
                 -->  a = [1, 2, 3]
                      b = a
                      a[2] = 9   # 이렇게 되면 a와 b 둘다 [1, 2, 9]를 가진다. Why? 같은 리스트를 참조하고 있기 때문이다.

                      따라서 b = a.copy() 를 b =a 대신 적어 리스트 자체를 복제하게 되면 b는 [1, 2, 3] a는 [1, 2, 9]가 된다.
      (3) in : 리스트, 튜플에 포함된 값을 측정
      (4) sort : 리스트를 정렬하기 위한 방법. 숫자는 오름차순으로 문자는 알파벳 순서이다.
                 --> 방식은 2가지 존재 (1) sorted 함수 : 리스트,튜플을 정렬해 바로 반환시켜준다...그러나 본래값은 변화가 없다
                                       (2) sort 매서드 : .sort() 정렬하고 반환하지는 않는다... 그러나 본래의 값을 바꾸어준다.
      (5) print : 정보를 출력 영역인 콘솔에 표시.
                  --> print("안녕"), print(False), print("Hello", 3, False) 출력가능 부울값은 따옴표 넣지 않는다
                  (1) % 연산자를 사용하는 방법 : %s(문자열), %d(10진수), %f(10진 부동소수점), %ox(16진수)가 존재.
                      %s %s % ("안녕", "나는 서울살아") 이러한 방식으로 사용
                      pint( %s %s % ("안녕", "나는 서울살아"))  print가 붙으면 .py파일 실행시 보인다. 그렇지 않다면 보이지 않는다
                      IDEL 방식에서는 둘다 보이나 print가 없다면 ' '안에 보이고 print는 깔끔하게 보인다.
                  (2) format 매서를 사용하는 방법 : "{1} {0}".format("문자","문자2") 를하면 문자2 문자가 출력이 된다...print사용 여부는 % 연산자와 동일하다
