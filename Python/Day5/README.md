## 공부한 내용

1. 부울값 이외의 값 : 조건식 위치에 값 기술 시 성립 여부가 결정이 된다... 
                      조건식 수치가 0, 문자열이 비거나 ' 혹은 "로 구성, 리스트와 튜플이 빈 경우는 False가 된다.
                      그 이외의 경우는 True가 되어 if아래 명령식을 바로 수행을 한다.
2. 삼항 연산자 : if-else를 이용하여 조금 다른게 조건식을 나타내는것
                 EX) 다소 귀찮은 감이 있다
                 if a>10:      
                     x = 10
                 else :
                     x = 20
                 따라서 이를 a = 10 if a>10 else 20 --> 이렇게도 표현이 가능하다. c,c++, java등에서 사용하는 ? : 연산과 동일한 의미이다.
3.while : 반복(루프) 처리를 하기 위한 명령
          EX) while 조건식 : 
                명령1
                명령2
              명령3
              --> 조건식이 True라면 명령1과 명령2를 실행하고 False가 되면 명령3으로 탈출한다.
4.for : 반복 처리를 위한 명령. index없이도 값들을 차례대로 꺼낼수 있다. 이것이 while 문과의 차이다.
        # 이러한 코드로 진행이 된다.
        for 변수 in 반복할 대상:
            실행할 코드
5.range : for문 사용할 때도 index를 이용하여 값을 찾는 명령. 
          #for과 비슷하나 조금 다름
          for 변수 in range(반복할 수)
          # 2가지 사용법
          range(최댓값) : 0부터 1씩 증가하나 최대는 포함하지 않는다.
          range(시작값, 최댓값) : 시작값부터 최댓값 직전까지 1씩 증가.
          range(시작값, 최댓값, 스텝) : 시작값부터 최댓값 직전까지 스텝에 맞춰 증가... 감소도 가능하다.
6. break와 continue : while과 for 등의 반복을에서 루프를 벗어나거나 도중에 제일 앞으로 되돌아가는 명령.
                      --> break는 루프를 탈출, continue는 루프를 건너 뛰고 다시 처음부터 시작하는 경우에 사용.
