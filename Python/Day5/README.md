## 공부한 내용

1. 부울값 이외의 값 : 조건식 위치에 값 기술 시 성립 여부가 결정이 된다... 
                      조건식 수치가 0, 문자열이 비거나 ' 혹은 "로 구성, 리스트와 튜플이 빈 경우는 False가 된다.
                      그 이외의 경우는 True가 되어 if아래 명령식을 바로 수행을 한다.
2. 삼항 연산자 : if-else를 이용하여 조금 다른게 조건식을 나타내는것
                 EX) 다소 귀찮은 감이 있다
                 if a>10:      
                     x = 10
                 else :
                     x = 20
                 따라서 이를 a = 10 if a>10 else 20 --> 이렇게도 표현이 가능하다. c,c++, java등에서 사용하는 ? : 연산과 동일한 의미이다.
3.while : 반복(루프) 처리를 하기 위한 명령
          EX) while 조건식 : 
                명령1
                명령2
              명령3
              --> 조건식이 True라면 명령1과 명령2를 실행하고 False가 되면 명령3으로 탈출한다.
